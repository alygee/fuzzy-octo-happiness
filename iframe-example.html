⚠️ Важно: Этот файл нужно открывать через локальный HTTP сервер, а не напрямую через file:// протокол!
Используйте: npm run preview или python3 -m http.server 8000, затем откройте http://localhost:8000/iframe-example.html

Примеры встраивания Inssmart Form через iframe

Пример 1: Фиксированная высота

Код:

<iframe 
  src="./dist/iframe/index.html"
  width="100%"
  height="800"
  frameborder="0"
  style="min-height: 800px;"
></iframe>

Пример 2: Адаптивная высота (с JavaScript)

Код:

<iframe 
  id="inssmart-iframe-adaptive"
  src="./dist/iframe/index.html"
  width="100%"
  frameborder="0"
  style="display: block; min-height: 600px;"
></iframe>

<script>
  const iframe = document.getElementById('inssmart-iframe-adaptive');
  
  window.addEventListener('message', function(event) {
    if (event.data && event.data.type === 'resize' && event.data.height) {
      iframe.style.height = event.data.height + 'px';
    }
  });
</script>

Пример 3: Полная ширина и высота

Код:

<div style="width: 100%; height: 900px;">
  <iframe 
    src="./dist/iframe/index.html"
    width="100%"
    height="100%"
    frameborder="0"
  ></iframe>
</div>
