# Инструкция по переносу дизайна из Figma

Это руководство поможет вам перенести дизайн-систему из Figma в проект.

## 1. Экспорт цветов

### Шаг 1: Определите цветовую палитру в Figma
1. Откройте ваш дизайн в Figma
2. Найдите все используемые цвета (обычно они находятся в стилях или в компонентах)
3. Запишите значения цветов

### Шаг 2: Конвертация цветов в HSL
Figma использует разные форматы цветов. Для проекта нужен формат HSL.

**Если цвет в RGB:**
- Используйте онлайн-конвертер или формулу: `hsl(h, s%, l%)`
- Пример: `rgb(59, 130, 246)` → `hsl(217, 91%, 60%)`

**Если цвет в HEX:**
- Используйте конвертер: https://www.hexcolortool.com/
- Пример: `#3b82f6` → `hsl(217, 91%, 60%)`

### Шаг 3: Обновление CSS переменных
Откройте файл `src/index.css` и обновите значения в секции `:root`:

```css
:root {
  --primary: 217 91% 60%; /* Ваш основной цвет из Figma */
  --primary-foreground: 210 40% 98%; /* Контрастный цвет для текста */
  
  /* Повторите для всех цветов */
}
```

**Важно:** Значения HSL должны быть без единиц измерения и скобок, только числа разделенные пробелами.

## 2. Экспорт типографики

### Шаг 1: Определите шрифты в Figma
1. Выберите текстовый элемент в Figma
2. Посмотрите свойства шрифта (семейство, размер, вес, высота строки)

### Шаг 2: Обновление конфигурации Tailwind
Откройте `tailwind.config.js`:

```js
theme: {
  extend: {
    fontFamily: {
      sans: ['Inter', 'system-ui', 'sans-serif'], // Замените на ваш шрифт
    },
    fontSize: {
      'xs': ['0.75rem', { lineHeight: '1rem' }],
      'sm': ['0.875rem', { lineHeight: '1.25rem' }],
      // Добавьте размеры из Figma
    },
    fontWeight: {
      // Добавьте веса шрифтов из Figma
    }
  }
}
```

### Шаг 3: Подключение шрифтов
Если используете кастомные шрифты, добавьте их в `index.html` или `src/index.css`:

```css
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap');
```

## 3. Экспорт компонентов

### Кнопки
1. Найдите стили кнопок в Figma
2. Откройте `src/components/ui/button.tsx`
3. Обновите классы Tailwind под дизайн:
   - Цвета: используйте CSS переменные
   - Размеры: обновите в `buttonVariants`
   - Скругления: измените `rounded-md` на нужное значение

### Поля ввода
1. Откройте `src/components/ui/input.tsx`
2. Обновите стили под дизайн из Figma:
   - Высота, отступы, границы
   - Цвета фона и текста
   - Состояния (focus, disabled)

### Карточки
1. Откройте `src/components/ui/card.tsx`
2. Обновите:
   - Тени (`shadow-sm`)
   - Скругления (`rounded-lg`)
   - Отступы (`p-6`)

## 4. Экспорт иконок

### Использование Lucide React
Проект использует библиотеку `lucide-react`. Если нужны другие иконки:

1. Найдите иконку в Figma
2. Проверьте, есть ли она в Lucide: https://lucide.dev/icons/
3. Если нет, экспортируйте SVG из Figma и создайте компонент:

```tsx
// src/components/icons/CustomIcon.tsx
export function CustomIcon({ className }: { className?: string }) {
  return (
    <svg className={className} /* ... */>
      {/* SVG из Figma */}
    </svg>
  )
}
```

## 5. Адаптивность

### Breakpoints
Проверьте breakpoints в Figma и сравните с Tailwind:
- `sm`: 640px
- `md`: 768px
- `lg`: 1024px
- `xl`: 1280px
- `2xl`: 1536px

Если нужны другие значения, обновите `tailwind.config.js`:

```js
theme: {
  screens: {
    'custom': '900px',
  }
}
```

## 6. Spacing и размеры

### Экспорт значений из Figma
1. Используйте инструмент измерения в Figma
2. Запишите все отступы, размеры элементов
3. Обновите компоненты, используя Tailwind классы:
   - `p-4` = 1rem (16px)
   - `p-6` = 1.5rem (24px)
   - и т.д.

Или добавьте кастомные значения в `tailwind.config.js`:

```js
theme: {
  extend: {
    spacing: {
      '18': '4.5rem',
      'custom': '120px',
    }
  }
}
```

## 7. Анимации и переходы

Если в Figma есть анимации:
1. Определите тип анимации (fade, slide, scale)
2. Добавьте в `tailwind.config.js`:

```js
theme: {
  extend: {
    keyframes: {
      'fade-in': {
        '0%': { opacity: '0' },
        '100%': { opacity: '1' },
      }
    },
    animation: {
      'fade-in': 'fade-in 0.3s ease-in-out',
    }
  }
}
```

## 8. Проверка результата

После всех изменений:
1. Запустите `npm run dev`
2. Сравните результат с дизайном в Figma
3. Внесите корректировки при необходимости

## Полезные инструменты

- **Конвертер цветов**: https://www.hexcolortool.com/
- **Figma to CSS**: Используйте плагины Figma для экспорта CSS
- **Tailwind IntelliSense**: Установите расширение для VS Code для автодополнения

## Пример полного переноса

1. **Цвета**: Основной цвет в Figma `#3b82f6` → `hsl(217, 91%, 60%)` → `--primary: 217 91% 60%`
2. **Шрифт**: Inter, 16px, 400 → `font-sans text-base font-normal`
3. **Кнопка**: Высота 44px, скругление 8px → `h-11 rounded-lg`
4. **Отступы**: 24px между элементами → `space-y-6`
